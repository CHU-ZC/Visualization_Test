# 钢铁与雪：苏德战争全景推演

> 讲述东线战场——苏德战争历史的网页端互动可视化平台

---

## 📖 项目简介

《钢铁与雪》是一个基于 Web 的交互式历史可视化项目，以 **东线战场（1941–1945 苏德战争）** 为核心叙事主线，通过动态地图、战役档案、统计图表与多媒体音效，还原这场人类历史上规模最大、伤亡最惨重的现代化战争。

---

## ✨ 核心功能

### 🗺️ 交互式战争推演地图

- **9 个关键战役节点**，按时间线依次展开：巴巴罗萨计划 → 列宁格勒围城 → 莫斯科保卫战 → 斯大林格勒战役 → 库尔斯克会战 → 基辅战役 → 巴格拉季昂行动 → 维斯瓦河-奥得河战役 → 柏林会战
- **进军/反攻路线可视化**：轴心国进军路线（黑色）与同盟国反攻路线（红色）动态绘制
- **关键坐标标注**：核心战役坐标（红）与胜利代表坐标（黄）
- **比例尺与罗盘**：内置东欧地图比例尺及指北针

### 📊 全局统计面板

- **统帅与核心将领**：希特勒、斯大林及双方名将介绍
- **累计动员兵力规模**：轴心国 vs 苏联的兵力对比（ECharts 图表）
- **惨烈伤亡统计**：双方阵亡、失踪数据可视化
- **结果与意义**：红旗插上柏林国会大厦的历史定论

### 🎬 沉浸式视听体验

- **入场动画**：德国 vs 苏联旗帜对峙画面 + GSAP 淡出过渡
- **背景音乐**：主题旋律与风雪环境底噪双音轨，支持独立开关
- **战役音效**：关键节点触发爆炸音效
- **阶段文字叠加层**：电影化叙事字幕

### 📝 详细战役档案

- 每个战役节点均可点击查看**图文档案弹窗**
- 包含历史照片、战况详情、历史回响
- 一键跳转 **维基百科** 深入阅读
- **历史总结与启示**模态框，回顾全局与和平警示

## 🚀 本地运行

### 方式一：直接打开

1. 克隆仓库

   ```bash
   git clone git@github.com:CHU-ZC/Visualization_Test.git
   ```

2. 用浏览器直接打开 `index.html`

### 方式二：本地服务器（推荐）

由于项目包含音频/视频资源，建议通过本地服务器运行以避免跨域问题：

```bash
# 使用 Python
cd Visualization_Test
python -m http.server 8080

# 或使用 Node.js
npx serve .
```

然后访问 `http://localhost:8080`
